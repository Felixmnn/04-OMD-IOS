# OMD App Privacy Policy Page - extended

## Overview

This file contains the privacy policy page for the OMD app. It includes privacy-related aspects necessary for the review process and submission of the app to the iOS and Android stores.

## Key Features

- The document addresses all privacy-related requirements for app store submission.
- It is written in a simple HTML format.

The privacy policy ensures compliance with app store regulations and provides transparency about data handling practices for users of the OMD app.
